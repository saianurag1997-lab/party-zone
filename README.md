# 🎉 Party Zone — Multiplayer Drinking Game Hub

Real-time party game app. Host on your laptop, players join from their phones on the same WiFi.

## 🚀 Setup (2 minutes)

```bash
cd party/server
npm install
node server.js
```

You'll see:
```
🎉 Party Game Server running!
📱 Share this with your friends:
   http://192.168.x.x:3000
```

## 📱 How Players Join

1. Everyone must be on the **same WiFi** as your laptop
2. Host opens `http://YOUR_IP:3000` and clicks **"Host a Game"**
3. A **QR code + room code** appears on the lobby screen
4. Friends scan the QR or type the URL on their phone
5. They click **"Join a Room"** and enter the 4-letter code
6. Host clicks **Start Game** when everyone's in!

## 🎮 Games Included

| Game | How it works |
|------|-------------|
| 🍺 Never Have I Ever | Tap if you've done it — those who have, DRINK! |
| 🤔 Would You Rather | Everyone votes A or B, see the split live |
| 👉 Most Likely To | Vote who in the group is most likely — winner gets a point (and maybe a drink) |
| 🧠 Drunk Trivia | Multiple choice — correct answers earn points |
| 🌶️ Hot Takes | Agree or disagree with spicy statements |
| ⚡ Rapid Fire | Timed challenges and social dares |
| 🍾 Spin the Bottle | Digital spin — whoever it lands on gets a dare |

## 🔧 Tips

- Keep everyone on the **same WiFi network**
- Works best on **Chrome** or **Safari** on mobile
- Up to **12 players** per room
- The host controls the pace with "Next Card"
- Leaderboard updates live as points are awarded
